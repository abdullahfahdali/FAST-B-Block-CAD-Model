# FAST B-Block CAD Model

A detailed AutoCAD-based architectural model of FAST-NUCES B Block developed for engineering design, visualization, and drafting practice.

## Project Overview

This project involved creating a digital model of FAST-NUCES B Block using AutoCAD. The objective was to improve CAD drafting skills, architectural visualization, and engineering design understanding through accurate modeling and layout development.

## Tools Used

- AutoCAD

## Features

- Detailed floor plan development
- Architectural modeling
- Engineering visualization
- Accurate layout representation
- CAD drafting and design
